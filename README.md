# Ranjan Kumar - Portfolio

A modern, responsive portfolio website built with React, Vite, and Tailwind CSS. Showcasing my work as a React Native & Frontend Developer at Flipkart.

## Live Demo

🔗 [View Portfolio](https://ranjankumar.vercel.app)

## Features

- **Modern UI/UX** - Glassmorphism design with smooth animations
- **Fully Responsive** - Works on all devices (mobile, tablet, desktop)
- **Dark Theme** - Eye-friendly dark mode design
- **Interactive Elements** - Custom cursor glow, particle background, animated sections
- **Performance Optimized** - Fast loading with Vite build system

## Tech Stack

- **Framework:** React 18
- **Build Tool:** Vite 5
- **Styling:** Tailwind CSS 3
- **Animations:** Framer Motion, GSAP
- **Icons:** Lucide React

## Sections

- Hero with animated text
- About Me with profile & highlights
- Professional Experience timeline
- Skills & Expertise with progress bars
- Featured Projects showcase
- Achievements & Education
- Licenses & Certifications
- Testimonials
- Contact form with social links

## Getting Started

### Prerequisites

- Node.js 18+ (recommended: 20.x)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/ranjanj17/portfolio.git
cd portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build for Production

```bash
npm run build
```

The build output will be in the `dist` folder.

### Preview Production Build

```bash
npm run preview
```

## Deployment

This project is configured for easy deployment on:

- **Vercel** (Recommended)
- **Netlify**
- **GitHub Pages**

### Deploy to Vercel

1. Push code to GitHub
2. Import project in [Vercel](https://vercel.com)
3. Vercel auto-detects Vite and deploys

### Deploy to Netlify

1. Push code to GitHub
2. Import project in [Netlify](https://netlify.com)
3. Build command: `npm run build`
4. Publish directory: `dist`
